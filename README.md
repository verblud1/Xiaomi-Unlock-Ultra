Установите (если не установлен):
    Bash

    sudo apt update && sudo apt install screen -y

    Создайте новую сессию (назовем её monitor):

Bash

   screen -S monitor

Перед вами откроется точно такой же терминал, но теперь он защищен.

    Запустите ваш скрипт:

Bash

   python3 server_scheduler.py

    Отключитесь от сессии (Detach):
    Нажмите комбинацию клавиш: Ctrl + A, а затем сразу клавишу D.
    Вы вернетесь в свой основной терминал SSH, а сверху появится надпись [detached]. Теперь можно закрывать консоль — скрипт работает.

    Как вернуться и проверить консоль позже:
    Когда вы снова подключитесь к серверу по SSH, введите:
    Bash

    screen -r monitor

    Вы снова увидите вывод своего скрипта. Если скрипт уже отработал, сессия закроется сама.
