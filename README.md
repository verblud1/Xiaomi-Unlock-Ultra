# Xiaomi Unlock Ultra

**Инструмент для автоматической точно-таймингованной подачи заявки на разблокировку загрузчика Xiaomi.**

Скрипт синхронизируется с NTP-серверами, прогревает TLS-сессии и отправляет POST-запрос в API Xiaomi в момент открытия окна — ровно в 00:00:00 по Пекину — с учётом сетевой задержки и jitter.

## Как это работает

1. **DNS Pinning** — резолвит `sgp-api.buy.mi.com` один раз и использует IP напрямую, исключая задержку DNS в момент выстрела.
2. **NTP-синхронизация** — опрашивает несколько серверов, отбрасывает выбросы по trimmed mean и получает точное пекинское время.
3. **Прогрев сессий** — до выстрела устанавливает TCP-соединение и TLS-хендшейк с keep-alive, чтобы к моменту запроса соединение уже было живым.
4. **Замер латентности** — 5 пингов до API, вычисляет медиану и jitter, на их основе рассчитывает поправку.
5. **Адаптивный сдвиг** — отправляет запрос с опережением, равным рассчитанной задержке, так что пакет прибывает на сервер ровно в полночь.
6. **Многократный TLS refresh** — за 8s, 5s и 2s до выстрела обновляет keep-alive, чтобы соединение не упало.
7. **Многопоточный выстрел** — каждый токен = отдельный поток; при одном токене запускаются два потока с разными сдвигами.

## Получение токена

1. Откройте браузер и перейдите на [c.mi.com/global](https://c.mi.com/global).
2. Войдите в свой аккаунт Mi Community.
3. Откройте DevTools (`F12`) → вкладка **Console**. (Если у вас браузер на базе Chromium, то вы можете вставить этот скрипт напрямую в поисковую строку)
4. Вставьте скрипт ниже и нажмите Enter:

```javascript
javascript :(function(){var token=document.cookie.match(/popRunToken=([^;]+)/);if(token){prompt("Copy the token:", token[1]);}else{alert("Token not found");}})()
```

> **Важно:** перед вставкой уберите пробел между `javascript` и `:`, если браузер его добавляет автоматически.

5. Скопируйте токен из появившегося окна.
6. Вставьте токен в файл `token.txt` — **каждый токен с новой строки**:

```
токен_аккаунта_1
токен_аккаунта_2
```

> **Примечание:** несколько токенов с одного аккаунта не дают преимущества — при исчерпании квоты на первом все остальные токены этого же аккаунта тоже получат `quota exceeded`.

## Запуск

```bash
source .venv/bin/activate
python3 main.py
```

Скрипт автоматически:
- синхронизируется с NTP
- прогреет сессии
- дождётся полуночи по Пекину
- выстрелит с точным таймингом

### Проверка связи с API

```bash
curl -sk -o /dev/null -w "%{http_code} | RTT: %{time_total}s\n" \
  https://sgp-api.buy.mi.com/bbs/api/global/user/bl-switch/state
```

Ожидаемый ответ: `200 | RTT: 0.XXXs`. Чем меньше RTT — тем лучше.

## Запуск на VPS в screen

Чтобы скрипт продолжал работать после закрытия SSH-сессии:

```bash
# Установить screen
sudo apt update && sudo apt install screen -y

# Создать именованную сессию
screen -S unlock

# Запустить скрипт внутри сессии
python3 main.py

# Отключиться от сессии (скрипт продолжает работать)
# Нажмите: Ctrl+A, затем D
```

Чтобы вернуться к сессии позже:

```bash
screen -r unlock
```

## Рекомендации по инфраструктуре

| Параметр | Рекомендация |
|---|---|
| **Локация VPS** | Сингапур или Гонконг |
| **Хостинги** | Избегайте Azure, AWS, GCP — их IP могут быть заблокированы Xiaomi |
| **VPN (если нет VPS)** | Платный, с серверами в Сингапуре / Гонконге |
| **RTT до API** | < 100ms — отлично, 100–300ms — приемлемо, > 500ms — нужен VPN |

**NTP-серверы для Сингапура** (уже вшиты в скрипт):

```
time.cloudflare.com   # anycast, ~1ms
sg.pool.ntp.org
asia.pool.ntp.org
cn.pool.ntp.org
ntp.aliyun.com
ntp.tencent.com
pool.ntp.org
```

**NTP-серверы для России:**

```
ntp2.vniiftri.ru
0.ru.pool.ntp.org  /  1.ru.pool.ntp.org  /  2.ru.pool.ntp.org  /  3.ru.pool.ntp.org
time.cloudflare.com
pool.ntp.org
```

## Подготовка устройства

Вне зависимости от ответа Xiaomi после полуночи по Пекину, выполянем следующие шаги.

**Шаг 1 — привязка аккаунта:**

> Настройки → Расширенные настройки → Для разработчиков → Статус Mi Unlock → **Связать аккаунт с устройством**

**Шаг 2 — если привязка не проходит:**

> Настройки → Mi аккаунт → Выйти из аккаунта → Перезагрузить устройство → повторить Шаг 1

**Шаг 3 — разблокировка через MiUnlock:**

Перейдите в MiUnlock и нажмите «Разблокировать». Если программа просит привязать аккаунт повторно, хотя в настройках написано «Привязан»:
- перезагрузите телефон
- запустите MiUnlock снова
- при необходимости попробуйте другую версию MiUnlock

> **Рекомендуется:** привязать номер телефона к аккаунту Xiaomi.

## Устранение проблем

**`token.txt не найден`** — убедитесь, что файл находится в той же папке, что и скрипт.

**`NTP offset слишком большой`** — системные часы сильно расходятся. Синхронизируйте время на VPS: `sudo ntpdate pool.ntp.org`.

**Высокий RTT (> 500ms)** — скрипт предупредит и снизит коэффициент компенсации. Используйте VPS ближе к Сингапуру.
